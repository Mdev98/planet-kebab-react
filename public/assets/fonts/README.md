# Fonts Directory

This directory should contain the proprietary font files from the original Planet Kebab application.

## Required Font Files

### Janda Manatee Solid
- `JandaManateeSolid.woff`
- `JandaManateeSolid.woff2`

### Gilroy Family
- `Gilroy-Medium.woff`
- `Gilroy-Medium.woff2`
- `Gilroy-Regular.woff`
- `Gilroy-Regular.woff2`
- `Gilroy-Thin.woff`
- `Gilroy-Thin.woff2`
- `Gilroy-ExtraBold.woff`
- `Gilroy-ExtraBold.woff2`

## How to Add Fonts

1. Obtain the font files from the original `Mdev98/planet-kebab-app` repository
2. Copy all `.woff` and `.woff2` files to this directory
3. The CSS in `src/App.css` already references these fonts with the correct paths
